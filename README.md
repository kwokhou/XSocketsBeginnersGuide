# Dont want to read the reademe.txt :)
Well, then just do this...
- Create a new MVC3 project
- Run "Install-Package XSockets"
- Add the Geolocation.cs to the XSocketHandler project (generated by the installer)
- Add default-3.htm to the root of your webApp.
- Set default-3.htm to be your startpage

- Compile and hit F5
